## Qual a diferença entre biblioteca e framework? 

As bibliotecas é um conjunto de códigos que vão solucionar um problema, várias bibliotecas são disponibilizadas de forma gratuita e livre. Por exemplo, a biblioteca Math, ela nos auxilia a realizar operações matemáticas, então não é necessário montarmos a solução, é só chamar a biblioteca e informar o que você deseja fazer.

Já os frameworks, eles nos dão uma base para construir a nossa aplicação, eles facilitam nosso trabalho, nos dando uma parte dele pronto e nos só o completamos.

Resumindo, uma biblioteca é o código de outra pessoa que você chama para facilitar na execução do seu código, enquanto um framework é o código de outra pessoa que chama o seu código.
